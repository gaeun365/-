# 🛡️ 프로젝트명: GuardUs (가드너)
> **사랑을 가장한 금융 사기, 로맨스 스캠으로부터 당신을 지킵니다.**

## 1. 해결하고자 하는 문제 (Problem)
* **메신저 사기 급증:** 카카오톡, 텔레그램 등에서 발생하는 로맨스 스캠 피해액 매년 증가.
* **심리적 고립:** 피해자는 사랑에 빠진 상태라 주변의 조언을 듣지 않는 경향이 있음.

## 2. 핵심 솔루션 (Solution)
### AI 보안 키보드 앱
사용자가 별도의 앱을 켤 필요 없이, 평소처럼 카톡을 할 때 키보드 단에서 즉시 위험을 감지합니다.

* **실시간 텍스트 스캔:** "송금", "코인 투자", "세금 문제" 등 사기꾼의 단골 멘트 감지.
* **이미지 팩트체크:** 상대방이 보낸 사진을 꾹 누르면 인터넷상 도용된 사진인지 즉시 검색.
* **강제 쿨타임:** 위험 감지 시 송금 앱으로 넘어가는 버튼을 10초간 잠금 (이성적 판단 유도).

## 3. 기술 스택 (Tech Stack)
* **Frontend:** React Native (iOS/Android 통합 개발)
* **AI:** Python, TensorFlow Lite (온디바이스 텍스트 분석)
* **Database:** Firebase (피해 사례 DB 연동)
