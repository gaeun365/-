# 🛡️ GuardUs (가드너)
**"사랑 뒤에 숨은 사기를 기술로 찾아냅니다."**

## 1. 프로젝트 개요
로맨스 스캠 피해 예방을 위한 온디바이스 AI 보안 키보드 솔루션입니다.

## 2. 주요 기능
- **실시간 텍스트 분석:** 대화 중 사기 유도 패턴 감지
- **이미지 역검색:** 프로필 및 일상 사진 도용 여부 확인
- **행동 넛지:** 위험 감지 시 송금 앱 전환 10초 지연

## 3. 사회적 임팩트
- 연간 OO억 원 규모의 로맨스 스캠 피해액 감소
- 디지털 취약 계층(시니어 등)의 안전한 소통 환경 조성
