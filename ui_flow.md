# 서비스 이용 시나리오 (User Flow)

1. **앱 설치:** 사용자가 '가드너' 앱 설치 후 기본 키보드로 설정.
2. **평소 대화:** 카카오톡 등에서 평소와 같이 대화 진행 (키보드 상단에 '안심' 아이콘 표시).
3. **위험 감지:** 상대방이 "수익률 좋은 코인이 있다"는 메시지를 보냄.
4. **즉각 알림:** 키보드 색상이 노란색으로 변하며 "투자 유도 주의" 문구 표시.
5. **이미지 검색:** 상대방 프로필 사진을 복사하자 "도용된 사진(Match 100%)" 팝업 발생.
6. **피해 방지:** 사용자가 송금을 포기하고 대화 종료.
